# Sign In / Sign Up UI

A clean, animated authentication UI built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

> Built from scratch as part of my self-taught front-end development journey.

---

## ✨ Features

- Smooth slide/fade transitions between Sign In and Sign Up panels
- Form validation with inline error feedback
- Responsive layout — works on mobile and desktop
- Dark mode support via `prefers-color-scheme`
- Accessible: keyboard navigable, semantic HTML, visible focus rings

---

## 🛠 Tech Stack

| Layer | Tech |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, animations) |
| Logic | Vanilla JavaScript (ES6+) |

---

## 🚀 Getting Started

```bash
git clone https://github.com/christopher0301-cloud/SignIn-SignUp.git
cd SignIn-SignUp
# Open index.html in your browser — no build step required
```
